About the App

BetView simplifies the process of tracking and analyzing betting activities, enabling users to make informed decisions. It wrote it using Python and the Django web framework. I chose to render HTML/CSS server-side with Python scripts, avoiding the use of a JavaScript framework. The app utilizes Bootstrap for responsive design and PostgreSQL for robust data handling. Interactive graphs are generated with Plotly, a leading Python data visualization library.
